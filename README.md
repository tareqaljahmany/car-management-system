# Car Management System (CLI)
A Python command-line application to manage cars.
## Features
- Add new cars
- View all cars
- Modify car information
- Input validation and error handling
- Object-Oriented Programming (OOP)
## Project Structure
- `main.py` → Main program that runs the CLI
- `validators.py` → Functions for input validation
## How to Run
   git clone https://github.com/tareqaljahmany/car-management-system.git
   
   cd car-management-system
   
   python main.py
